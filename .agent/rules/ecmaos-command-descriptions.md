# EcmaOS Command Descriptions

## File Operations
- **fetch**: Download file from URL
- **download**: Download and save resource
- **load**: Load module or script
- **cat**: Display file contents
- **touch**: Create empty file or update timestamp
- **rm**: Remove file or directory
- **cp**: Copy files or directories
- **mv**: Move or rename files
- **edit**: Open file in editor

## Directory Operations
- **ls**: List directory contents
- **cd**: Change directory
- **pwd**: Print working directory
- **mkdir**: Create directory

## System Information
- **env**: Display environment variables
- **df**: Show disk space usage
- **du**: Estimate file space usage
- **ps**: List running processes
- **free**: Display memory usage

## Process Management
- **kill**: Terminate process

## Disk Management
- **chkdisk**: Check disk for errors
- **format**: Format storage device
- **mount**: Mount filesystem
- **umount**: Unmount filesystem

## Entertainment
- **snake**: Play snake game
- **video**: Play video file
- **play**: Play media file
- **screensaver**: Activate screensaver

## Notes
- All descriptions should be concise (under 40 characters)
- Focus on primary use case
- Use imperative mood (e.g., "Display" not "Displays")
